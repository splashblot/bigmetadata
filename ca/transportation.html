UPDATE {table_name}
  SET {new_numeric_column} =
    OBS_GetMeasure(
      CDB_LatLng(43.6559499126, -79.3796539307),
      'ca.statcan.cols_nhs.t017c002_f'
    );

Median commuting duration (female) is only available for point lookups.
